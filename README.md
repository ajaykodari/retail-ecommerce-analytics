## üìå Project Overview

A complete, industry-level retail analytics solution built on a normalized MySQL database with 5 tables, 50 customers, 30 products, and 100+ orders across 2022‚Äì2024. The pipeline extracts raw data using SQL, cleans and engineers features using Python/Pandas, and delivers insights through both a live interactive web dashboard and a professional Power BI report.

**Key Business Questions Answered:**
- Which products and categories drive the most revenue and profit?
- Who are our highest-value customers (CLV)?
- How is revenue trending YoY?
- What does our customer segmentation look like (RFM)?
- Where are profit margins highest?

## üî¢ SQL ‚Äî Key Queries

### Master Sales Fact Table
```sql
SELECT
    o.order_id, o.order_date, o.ship_mode, o.region,
    c.customer_name, c.segment, c.city,
    p.product_name, p.category, p.brand,
    oi.quantity,
    ROUND(oi.quantity * oi.unit_price * (1 - oi.discount), 2) AS revenue,
    ROUND((oi.quantity * oi.unit_price * (1 - oi.discount))
          - (oi.quantity * p.cost_price), 2)                   AS profit
FROM orders o
JOIN customers c   ON o.customer_id = c.customer_id
JOIN order_items oi ON o.order_id  = oi.order_id
JOIN products p    ON oi.product_id = p.product_id
LEFT JOIN returns r ON o.order_id  = r.order_id;
```

### Customer Lifetime Value
```sql
SELECT
    c.customer_name,
    COUNT(DISTINCT o.order_id)                                      AS total_orders,
    ROUND(SUM(oi.quantity * oi.unit_price * (1 - oi.discount)), 2) AS clv_estimate
FROM customers c
JOIN orders o ON c.customer_id = o.customer_id
JOIN order_items oi ON o.order_id = oi.order_id
GROUP BY c.customer_id
ORDER BY clv_estimate DESC;
```

### RFM Segmentation Base
```sql
SELECT
    c.customer_id,
    DATEDIFF('2024-12-31', MAX(o.order_date)) AS recency_days,
    COUNT(DISTINCT o.order_id)                AS frequency,
    SUM(oi.quantity * oi.unit_price)          AS monetary
FROM customers c
JOIN orders o ON c.customer_id = o.customer_id
JOIN order_items oi ON o.order_id = oi.order_id
GROUP BY c.customer_id;
```

## üìê DAX Measures (Power BI)

```dax
-- Total Revenue
Total Revenue = SUMX(sales_fact, sales_fact[revenue])

-- YoY Revenue Growth
YoY Revenue Growth % =
    DIVIDE([Total Revenue] - [Revenue PY], [Revenue PY], 0) * 100

-- Customer Lifetime Value
Avg CLV = AVERAGE(customer_clv[clv_estimate])

-- Revenue MTD
Revenue MTD = CALCULATE([Total Revenue], DATESMTD(DateTable[Date]))
```

Full list of 20+ DAX measures in `powerbi/powerbi_dax_guide.md`
